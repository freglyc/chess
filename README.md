# Chess Application
Chess application coded functionally with the [vavr](https://www.vavr.io/) library.
Written as a self test of current ability. 

## Setup
Clone Repo:
```$xslt
https://github.com/freglyc/chess.git
```
Build: 
```$xslt
./gradlew build
```
